# E-Commerce Customer Intelligence & Revenue Optimization

An end-to-end data science project analyzing transactional e-commerce data to understand revenue drivers, segment customers, predict churn, and simulate A/B discount experiments.

---

##  Project Overview

This project explores:

- Revenue analysis and business insights
- RFM-based customer segmentation using KMeans
- Time-based churn prediction using Logistic Regression
- Statistical A/B testing simulation
- Revenue lift evaluation with hypothesis testing

---

##  Dataset

This project uses the Online Retail II dataset (UK-based e-commerce transactions).
Due to GitHub file size limits (>25MB), the dataset is not included in this repository.
You can download it from Kaggle:
https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset?resource=download
After downloading, place the file in the project root or data folder before running the notebook.

##  Key Highlights

- Revenue concentration identified among high-value customers
- Customers segmented into behavioral groups (Regular, VIP, At-Risk, etc.)
- Time-aware churn model built to avoid data leakage
- Class imbalance handled using weighted logistic regression
- A/B experiment simulated using:
  - Two-proportion z-test
  - Two-sample t-test
  - Bootstrap confidence intervals
